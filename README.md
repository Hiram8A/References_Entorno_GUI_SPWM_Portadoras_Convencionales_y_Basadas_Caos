# References_Entorno_GUI_SPWM_Portadoras_Convencionales_y_Basadas_Caos
Este trabajo presenta el desarrollo de un entorno gráfico interactivo (por sus siglas en inglés, GUI) en MATLAB para el análisis comparativo de técnicas SPWM, implementando cinco tipos de señales portadoras: triangular, sinusoidal, diente de sierra, aleatoria y caótica (generada mediante mapas logísticos).
